## 오늘 배운 것
- 시간복잡도: O(n log n)
- js로 sort처리하는걸 배웠다
```js
    targets.sort((a, b) => a[0] - b[0]) // 첫번재꺼 오름차순
    targets.sort((a, b) => b[0] - a[0]) // 첫번재꺼 내림차순
    targets.sort((a, b) => a[1] - b[1]) // 두번째꺼 오름차순
    targets.sort((a, b) => b[1] - a[1]) // 두번째꺼 내림차순
```
- 역시 그리디는 생각하는게 너무 힘들다
