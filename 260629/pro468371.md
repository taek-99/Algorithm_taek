- 처음 최소공배수를 통해 한사이클만 확인해야겠다는 접근하는데 시간이 좀 걸렸다
- 유클리드 호제법이 잘 기억 안나서 이번엔 그냥 math 라이브러리를 사용했다
- 매초마다 노란불을 어떻게 확인할까 고민을 많이했는데 나는 각 신호를 리스트로 만들어 0, 1, 2로 구분하여 매초 비교했는데 찾아보니 비효율적인거 같다
- 주기까지 다 구해놨으니 어디 범위에 있는지만 확인하면 됐다
```python 
        for G, Y, R in signals:
            cycle = G + Y + R
            pos = (t - 1) % cycle

            if not (G <= pos < G + Y):
                is_all_yellow = False
                break
```
