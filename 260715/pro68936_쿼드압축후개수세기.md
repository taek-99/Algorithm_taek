## 힘들었던 점
- visited를 생성하여 관리하고 싶은데 js로 visited를 만드는데 시간이 꽤 걸렸다
- 그거 외엔 n을 2로 나눠가면서 진행해야겠다는 생각은 빨리 들어서 금방 했던거 같다

## 오늘 배운 것
- 시간복잡도: O(n log n)
- visited 만드는 방법
```js
const visited = Array.from({ length: n }, () => Array(n).fill(false))

```
- from: n개만큼 배열 만들기
- fill: 배열 안에 같은 값 넣기
